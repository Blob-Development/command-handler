# command-handler
Command Handler Template, exclusive by **Blob Project.**<br>
• discord.js (v12.5.1)<br>
• Node.js (v14.x)<br>

# Configuration
- Run `npm install`
- Put your token inside `.env` file by replacing `token123123123` with the rest.
- Run `node .`
